학생 가이드

달란트 마을을 처음 사용하는 학생을 위한 안내입니다. 계정 신청부터 달란트 확인, 상품 구매까지 순서대로 알려드립니다. 비로그인 상태에서 소개 > 가이드를 누르면 이 학생 가이드가 열립니다.

📝

계정 신청

계정이 없으면 로그인 화면에서 등록 신청을 할 수 있습니다. 신청 후 관리자가 승인해야 로그인할 수 있습니다.

💰

내 달란트 확인

로그인하면 사용 가능·사용 대기·사용 완료·반환·누적 적립 달란트와 적립/사용/반환 내역을 확인할 수 있습니다.

🛒

상품 구매 신청

학생용 상품을 보고 구매 신청할 수 있습니다. 신청한 상품은 선생님이 순서대로 처리합니다.

📱

QR 달란트 수령

달란트 수령 메뉴에서 QR 코드를 카메라로 스캔하거나 코드를 입력하여 달란트를 받을 수 있습니다. 유효 날짜·시간과 위치 제한이 있는 QR은 조건을 만족할 때만 지급됩니다.

💬

질문하기

Q & A에서 자주 묻는 질문을 보고, 궁금한 내용을 질문으로 남길 수 있습니다.

🚀 처음 시작하기

  1. 메인 화면에서 로그인을 누릅니다.
  2. 계정이 없다면 로그인 화면의 계정 등록 신청을 누릅니다.
  3. 희망 아이디를 입력하고 중복확인을 합니다. 아이디는 영문, 숫자, 밑줄, 하이픈만 사용할 수 있습니다.
  4. 이름과 소속 부서를 선택하고 신청합니다.
  5. 승인이 끝나면 초기 비밀번호 1234로 로그인합니다.
  6. 첫 로그인 때 새 비밀번호로 바꾸면 사이트를 사용할 수 있습니다.
비밀번호는 8자 이상이어야 하고 영문과 숫자가 모두 들어가야 합니다. 초기 비밀번호 1234는 새 비밀번호로 사용할 수 없습니다.

2 내가 볼 수 있는 메뉴 중에서 1개 이상, 최대 10개까지 선택하고 저장을 누릅니다.
3 설정은 계정에 저장되어 다른 기기에서도 동일하게 표시됩니다.

🎨 테마 설정

일반 / 다크 모드

상단 메뉴 오른쪽의 테마 스위치로 일반 모드와 다크 모드를 바꿀 수 있습니다. 로그인한 사용자의 선택은 계정 설정에 저장되어 다른 기기에서도 이어지고, 로그아웃하면 기본 일반 모드로 돌아갑니다.

📱 모바일에서 사용하기

상단 메뉴 배치

스마트폰 등 좁은 화면에서는 상단 우측에 아래 순서로 버튼이 표시됩니다.

  1. 테마 스위치 — 일반/다크 모드 전환
  2. ☰ 메뉴 — 햄버거 버튼을 눌러 전체 메뉴 열기
  3. 로그아웃 — 현재 계정에서 로그아웃
  4. 이름 — 권한 이모지와 함께 로그인한 사용자 이름 표시
PC 화면보다 메뉴가 접혀 있으므로, 원하는 메뉴는 ☰ 버튼을 눌러 확인하세요.

⏱️ 세션 관리 (자동 로그아웃)

24시간 유휴 타임아웃

로그인한 뒤 24시간 동안 사이트를 사용하지 않으면 세션이 만료되어 자동으로 로그아웃됩니다. 만료 시 안내 메시지가 표시되고 로그인 화면으로 이동합니다.

활동 시 시간 초기화

페이지 조회, 클릭, 스크롤, 키 입력 등 사이트를 사용하는 활동이 있으면 24시간 타이머가 처음부터 다시 시작됩니다. 여러 탭을 열어 두어도 활동 시간은 공유됩니다.

🛒 상품 구매 흐름

  1. 상품 구매에서 원하는 상품을 고릅니다.
  2. 구매 신청을 누릅니다.
  3. 신청한 금액은 바로 차감되지 않고 사용 대기로 잡힙니다.
  4. 선생님이 실제 구매를 확정하면 달란트가 차감됩니다.
  5. 상품을 받으면 상태가 상품 지급으로 바뀝니다.

진행 단계

구매 신청 → 상품 준비 → 상품 구매 → 상품 지급

상품 카테고리와 구매 진행 상태 이름은 공통 코드 기준으로 표시됩니다. 상품 구매, 내 구매 상품, 선생님 구매 관리 화면에서 같은 단계는 같은 이름으로 보입니다.

📊 내 달란트 화면 보는 법

항목뜻
사용 가능 달란트 ⭐지금 바로 상품 구매에 사용할 수 있는 달란트입니다.
수령 대기 상품수 🎁구매가 진행 중이고 아직 상품을 받지 않은 건수입니다.
사용 대기 달란트 ⏳구매 신청은 했지만 아직 달란트가 차감되지 않은 금액입니다.
사용 완료 달란트 📉실제 상품 구매에 사용된 달란트만 표시됩니다. 교사가 잘못 지급한 달란트를 돌려받은 금액(반환)은 포함되지 않습니다.
반환 달란트 ↩️분홍색 요약 박스로 표시됩니다. 교사가 잘못 지급한 달란트를 돌려받은 총액입니다.
누적 적립 달란트 📈지금까지 받은 달란트의 총 합계입니다.

달란트 내역 유형

하단 달란트 내역 테이블에서는 각 건마다 유형 배지로 구분됩니다.

  • 적립 — 활동·QR 등으로 달란트를 받은 경우
  • 사용 — 상품 구매 등으로 달란트가 차감된 경우
  • 반환 — 교사가 잘못 지급한 달란트를 돌려받은 경우 (사용과 별도 표시)
지급 취소(반환) 이력의 내부 트랜잭션 ID는 설명란에 표시되지 않습니다.

상품 구매 진행 흐름

구매 신청 → 상품 준비 → 상품 구매 → 상품 지급

  • 구매 신청 : 학생이 상품을 선택하고 신청합니다. 이 단계에서 취소할 수 있습니다.
  • 상품 준비 : 선생님이 신청을 확인하고 상품을 준비합니다.
  • 상품 구매 : 상품 구매가 확정되면 달란트가 차감됩니다.
  • 상품 지급 : 상품을 실제로 전달받으면 완료됩니다.

❓ 자주 헷갈리는 상황

로그인이 안 돼요

아이디와 비밀번호를 다시 확인하세요. 계정 신청이 승인 전이면 승인 대기 안내가 표시됩니다.

구매했는데 달란트가 그대로예요

구매 신청만 한 상태라면 정상입니다. 선생님이 상품 구매를 확정하면 실제로 차감됩니다.

비밀번호를 잊었어요

담당 선생님이나 관리자에게 초기화를 요청하세요. 초기화 후 1234로 로그인하고 새 비밀번호로 바꾸면 됩니다.

화면 색이 달라졌어요

상단 테마 스위치에서 일반/다크 모드를 바꿀 수 있습니다. 로그인 상태에서는 선택한 테마가 계정에 저장됩니다.

갑자기 로그아웃됐어요

24시간 동안 사이트를 사용하지 않으면 세션이 만료되어 자동 로그아웃됩니다. 다시 로그인하면 됩니다. 사용 중에는 활동할 때마다 타이머가 초기화됩니다.

반환 달란트가 뭐예요?

선생님이 실수로 지급한 달란트를 되돌린 금액입니다. 내 달란트 화면의 분홍색 ↩️ 박스와 내역의 반환 배지로 확인할 수 있습니다.
