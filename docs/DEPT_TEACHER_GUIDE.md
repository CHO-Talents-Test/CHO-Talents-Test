# 부서 담당 교사 가이드

대상: `dept_teacher`, 60등급

부서 담당 교사는 담당 부서 안의 학생/교사, 달란트 지급, 상품 신청, Q&A 답변을 관리한다. 일반 교사가 없는 부서에서는 현장 달란트 지급과 학생 계정 지원을 함께 맡는다.

## 핵심 업무

| 업무 | 할 수 있는 일 |
|---|---|
| 사용자 관리 | 담당 부서 사용자 조회, 가입 신청 처리, 담당 학생 반 변경, 비밀번호 `1234` 초기화 |
| 달란트 관리 | 담당 부서 학생/교사 달란트 지급, 출석 빠른 지급, 상세 내역 조회 |
| 상품/구매 | 상품 등록/수정, 담당 부서 구매 신청 처리 |
| 구매 통계 | 담당 부서 구매 흐름 확인 |
| Q&A | 질문 답변, 댓글 작성, FAQ 등록 |

## 일반 교사가 없는 부서에서 해야 할 일

1. `달란트 > 달란트 관리`에서 담당 부서 학생 목록이 정상 표시되는지 확인한다.
2. 출석, 성경 읽기, 말씀 암송 등 실제 활동 달란트를 직접 지급한다.
3. `상품 > 구매 관리`에서 담당 부서 주문을 상품 준비, 상품 구매, 상품 지급 순서로 처리한다.
4. 계정 문제가 있으면 `관리 > 사용자 관리`에서 반 변경 또는 비밀번호 초기화를 처리한다.
5. Q&A 미답변 질문을 확인하고 담당 부서 관련 질문에 답한다.

## 제한

- 사용자 삭제와 부서 이동 처리는 부장 교사(80+) 이상 권한이 필요하다.
- 상품 삭제는 전도사님(90+) 이상 권한이 필요하다.
- 수동 달란트 적립은 관리자(100+) 기준으로 제한된다.
