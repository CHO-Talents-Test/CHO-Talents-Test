# 구매 담당 교사 가이드

대상: `purchase_teacher`, 70등급

구매 담당 교사는 부서 담당 교사와 같은 관리 메뉴를 사용하면서, 구매 관리와 상품 배지에서 전체 부서 주문을 처리할 수 있다.

## 핵심 업무

| 화면 | 해야 할 일 |
|---|---|
| `admin/purchases.html` | 전체 부서 주문 조회, 상태 변경, 일괄 상품 준비/구매 확정/지급 처리 |
| `admin/purchase-stats.html` | 전체/부서별/사용자별/유형별 구매 통계 확인 |
| `admin/shop.html` | 상품 가격, 카테고리, 이미지, 대상 확인 및 등록/수정 |
| 상단 상품 배지 | 전체 부서의 처리 가능한 주문 수 확인 |

## 구매 처리 순서

1. 구매 신청 상태 주문을 확인한다.
2. 실제 주문 접수 또는 준비가 시작되면 `상품 준비`로 변경한다.
3. 실제 구매가 완료되면 `상품 구매`로 확정한다. 이때 달란트가 실제 차감된다.
4. 상품 전달이 끝나면 `상품 지급`으로 완료한다.

## Slack 알림

- 신규 구매 신청은 신청자 소속 부서 채널로 전송된다.
- `구매 신청 -> 상품 준비` 전환만 상품 관리 채널로 전송된다.
- 상품 구매 확정, 상품 지급, 되돌리기, 취소는 현재 Slack 알림 대상이 아니다.
