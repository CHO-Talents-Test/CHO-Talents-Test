## 변경 사항

### 1. 모바일 반응형 개선 (`css/admin.css`)
- **900px 이하 (태블릿/모바일)**:
  - 네비게이션을 세로 방향으로 변경, 링크 줄바꿈 허용
  - 폰트 크기 축소 (링크 0.75rem, 브랜드 1rem)
  - 역할 배지 라벨 숨김 (이모지+이름만 표시)
- **640px 이하 (스마트폰)**:
  - 통계 카드 2열 그리드로 변경 (기존 auto-fit)
  - 카드 패딩/폰트 축소
  - 테이블 가로 스크롤 지원 (min-width: 600px)
  - 테이블 폰트/패딩 축소
  - 필터 버튼 축소
  - 모달 패딩 축소, 너비 95%
  - 컨테이너 패딩 축소

### 2. 로그 일괄 완료처리 (`admin/logs.html`)
- 미확인 ERROR+ 로그 섹션에 "📋 일괄 완료처리" 버튼 추가
- 클릭 시 사유 입력 prompt → 확인 → 전체 미확인 로그 일괄 처리
- 처리 완료 후 결과 알림 + 로그 새로고침
