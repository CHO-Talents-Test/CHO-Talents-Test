# 전도사님 가이드

대상: `evangelist`, 90등급

전도사님은 달란트 항목, QR, 상품 삭제 대기, Q&A 삭제, 부서 비활성화처럼 운영 정책에 직접 영향을 주는 작업을 담당한다.

## 특별 권한

| 영역 | 가능 작업 |
|---|---|
| 달란트 항목 | 학생용/교사용 항목 등록, 수정, 활성 상태 관리 |
| 달란트 QR | QR 생성, 수정 시 새 코드 재생성, 비활성화, 인쇄, 수령자 확인 |
| 상품 | 상품 삭제 대기 처리 |
| Q&A | 질문 삭제, 댓글 삭제 |
| 부서 | 부서 삭제/비활성화, 전체 부서 이동 처리 |

## QR 운영 순서

1. 지급 대상이 학생인지 교사인지 선택한다.
2. 달란트 항목을 선택하거나 직접 입력으로 설명과 금액을 입력한다.
3. 무제한/선착순, 지정일/기간/무기한, 반복 수령 여부를 설정한다.
4. 위치 제한이 필요하면 카카오맵 검색으로 장소와 반경을 지정한다.
5. 생성 후 QR 이미지를 인쇄하거나 화면으로 공유한다.
6. 운영이 끝난 QR은 비활성화하고 수령자 목록을 점검한다.

## 점검 포인트

- 달란트 항목의 지급 규칙과 설명이 적립 안내 화면의 문구와 맞는지 확인한다.
- 상품 삭제 전 진행 중인 주문이 없는지 확인한다.
- 부서 비활성화 전 소속 사용자와 가입 신청, 부서 이동 요청을 확인한다.
- Q&A 삭제는 민감한 내용, 중복, 잘못된 질문에 한정하고 FAQ 전환 가능성을 먼저 본다.
