# 부장 교사 가이드

대상: `chief`, 80등급

부장 교사는 전체 운영 현황을 확인하고 담당 관리 부서 중심으로 승인, 보고서, 버전 확인, 학생 일괄 등록, 달란트 반환을 수행한다.

## 특별 권한

| 권한 | 설명 |
|---|---|
| 학생 일괄 등록 | 엑셀 양식 업로드로 학생 계정을 한 번에 생성 |
| 달란트 반환 | 잘못 지급된 달란트를 사유 입력 후 반환 |
| 보고서/버전 확인 | 작업 보고서와 버전 이력 조회 |
| 운영 룰 문서 열람 | 페이지 권한 룰, 로그 작성 룰, Slack 알림 룰, 작업 이력 룰 열람 |

## 자주 하는 작업

1. `관리 > 학생 일괄 등록`에서 새 학생 계정을 등록한다.
2. `관리 > 사용자 관리`에서 가입 신청과 부서 이동 요청을 확인한다.
3. `달란트 > 달란트 관리` 상세 화면에서 잘못 지급된 달란트를 반환한다.
4. `상품 > 구매 통계`에서 부서별 주문 흐름을 확인한다.
5. `운영 > 보고서`, `운영 > 버전`에서 변경 내역을 확인한다.

## 제한

- 로그/작업 이력 화면 직접 조회는 관리자(100+) 권한이다.
- 페이지 접근/기능 설정 변경도 관리자(100+) 권한이다.
- QR 관리와 상품 삭제는 전도사님(90+) 이상 권한 기준이다.
