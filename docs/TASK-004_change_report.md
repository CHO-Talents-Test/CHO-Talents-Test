## 변경 사항

### 1. 버전 관리 시스템 도입
- `js/version.js` 신규 생성
  - `APP_VERSION` 객체: 현재 버전, 날짜, 전체 이력 관리
  - `getVersion()`, `getVersionHistory()` 함수
  - `renderVersionBadge()`: 모든 페이지 footer에 자동 버전 배지 표시
- 전체 20개 HTML 파일에 `version.js` 스크립트 태그 추가

### 2. 관리자 버전 이력 조회 페이지
- `admin/versions.html` 신규 생성
  - 현재 버전 배지 (버전번호, 타이틀, 업데이트 날짜)
  - 타임라인 형식의 버전 이력 조회
  - 버전별 변경 내용 목록
  - 최신 버전 강조 표시 (녹색)
- admin 네비게이션 전체에 "버전" 링크 추가

### 3. 보고서 체계 정비
