SeeLIS SQLite 뷰어

.db / .sqlite 파일을 브라우저에서 바로 열어 테이블·스키마를 살펴보고 SQL을 실행합니다. 아래 영역을 클릭하거나 파일을 끌어다 놓으세요.

📂 클릭하여 파일 선택  ·  또는 드래그 앤 드롭

⚠️ 이 뷰어는 SQLite 엔진(sql.js · WebAssembly)을 CDN에서 불러옵니다. 처음 파일을 열 때 인터넷 연결이 필요합니다. 한 번 로드되면 같은 세션에서는 다시 받지 않습니다. 데이터는 브라우저 메모리에서만 처리되며 원본 파일과 외부로 전송되지 않습니다.

메모리 DB에서 실행되어 원본 파일은 안전합니다. SELECT 외 쿼리(UPDATE/DELETE 등)도 실행 가능하나 변경은 메모리에만 반영됩니다.
