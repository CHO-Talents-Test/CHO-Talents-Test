교사 가이드

교사 계정으로 학생 달란트 지급, QR 달란트 수령, 대리 구매 신청을 하는 방법을 안내합니다. 소개 > 가이드는 로그인한 계정 권한에 맞는 가이드로 자동 연결됩니다.

🔑 처음 로그인할 때

  1. 관리자에게 계정을 받은 뒤 로그인합니다. 초기 비밀번호는 보통 1234입니다.
  2. 첫 로그인 화면에서 새 비밀번호를 입력합니다.
  3. 비밀번호는 8자 이상, 영문과 숫자를 포함해야 합니다.
  4. 메인 화면으로 이동하면 상단 메뉴에서 내 권한에 맞는 메뉴가 보입니다.
로그인 상태는 24시간 유휴 시 자동으로 만료됩니다. 페이지 조회, 클릭, 스크롤, 키 입력 등 사이트를 이용하는 활동이 있으면 만료 시간이 다시 24시간으로 초기화됩니다. 다른 탭으로 이동했다가 돌아올 때도 만료 여부를 다시 확인하며, 만료된 경우 안내 후 로그인 화면으로 이동합니다.
운영자가 활성화한 공지가 있으면 로그인 후 메인 화면에 공지 팝업이 표시됩니다. 다시 열지 않음을 누르면 해당 계정에는 같은 공지가 다시 표시되지 않습니다.

📌 교사가 주로 쓰는 메뉴

메뉴설명
달란트 관리담당 학생에게 달란트 항목을 선택하여 지급합니다. 출석 버튼으로 빠르게 지급하거나, 관리 드롭다운에서 달란트 지급/상세를 선택합니다. 테이블 헤더를 클릭하면 해당 열 기준으로 정렬할 수 있습니다.
달란트 수령QR 코드 스캔 또는 코드 입력으로 달란트를 수령합니다. 지정된 유효 시간과 위치 제한이 있으면 해당 조건을 만족해야 하며, 위치 권한이 차단된 기기에서는 알림창이 표시됩니다. 최근 수령 내역은 페이지당 항목 수를 선택할 수 있습니다.
상품 구매교사용 상품을 구매하거나 학생용 상품을 확인합니다. 교사는 학생 상품을 직접 구매할 수 없습니다.
대리 구매학생을 선택(이름·유형·부서·반 표시)하여 학생 명의로 상품 구매 신청을 합니다.
구매 관리부서 담당 교사 이상은 칸반보드 형태의 상태별 카드로 현황을 파악하고 권한 범위 안의 주문을 처리합니다. 구매 담당 교사는 모든 부서 주문을 처리할 수 있습니다.
구매 통계부서 담당 교사 이상은 전체/부서별/사용자별/유형별 구매 통계를 확인합니다. 교사/학생 분리 표시, 상세 드릴다운, 페이지당 항목 수 선택과 기본 정렬을 제공합니다.
내 달란트사용 가능 달란트, 수령 대기 상품수, 사용 완료·반환·예외 달란트 요약, 적립/예외/사용/반환 내역, 상품 수령 예정 현황을 확인합니다.
대시보드부서 담당 교사(60+) 이상 권한이면 관리 대시보드에 접근할 수 있습니다. 통계 카드와 이슈 로그 테이블을 확인합니다.

💰 학생에게 달란트 지급하기

  1. 상단 메뉴에서 달란트 > 달란트 관리로 이동합니다.
  2. 학생 탭에서 담당 학생을 찾습니다.
  3. 출석 항목은 학생 행의 출석 버튼으로 빠르게 지급할 수 있습니다.
  4. 다른 항목은 달란트 지급을 누르고 지급할 항목을 체크합니다.
  5. 지급 확정을 누르면 선택한 항목이 한 번에 반영됩니다.
  6. 상세를 누르면 해당 학생의 달란트 요약과 내역을 확인할 수 있습니다. 상세 팝업 상단에는 누적적립, 총사용, 총반환, 잔여가 구분되어 표시되고, 이력은 페이지 이동과 페이지당 항목 수 선택을 지원합니다.
항목 지급은 주간(월~일) 1회 제한을 따릅니다. 이미 지급된 항목은 완료 또는 취소 표시가 나오며, 예외 재지급은 전도사님(90+) 이상만 사유 입력 후 처리할 수 있습니다. 취소/반환은 같은 거래 ID 기준으로 동기화됩니다. 지급 취소 이력의 달란트 트랜잭션 ID는 관리자가 아닌 사용자에게는 표시되지 않습니다.

📊 달란트 내역 유형 (v3.48.0)

달란트 내역 테이블의 유형 열에는 다음 세 가지가 구분되어 표시됩니다.

  • 적립: 선생님이나 QR 등으로 받은 달란트
  • 사용: 상품 구매 등에 실제로 차감된 달란트
  • 예외: 주간 제한을 예외로 다시 지급한 건
  • 반환: 잘못 지급된 달란트를 되돌린 건 (사용과 별도 집계)

반환은 잘못 지급된 달란트를 사유 입력 후 되돌리는 기능으로, 부장 교사(80+) 이상 권한에서만 처리할 수 있습니다. 일반 교사는 내역에서 반환 건을 확인만 할 수 있습니다.

📋 작업 이력 확인

달란트 지급/취소 등 관리 작업 내역을 작업 이력에서 확인할 수 있습니다.

1 운영 → 작업 이력 메뉴로 이동합니다 (관리자 전용).
2 10개 카테고리 필터로 원하는 작업 유형만 조회할 수 있습니다.
3 작업 이력은 활동 로그 중 관리 작업만 모아 보여주며, 모든 작업 내역이 한글로 표시됩니다.

코드 입력

QR 코드 아래에 적힌 코드를 입력하고 코드 확인을 누릅니다. 관리자 외에는 입력값이 가려져 보입니다.

카메라 스캔

카메라 버튼을 누르고 QR 코드를 비추면 코드가 자동으로 입력되고 확인됩니다. 처리 결과 메시지는 카메라 영역 위에 표시됩니다.

⏰

유효 시간

지정일 QR은 날짜뿐 아니라 시작·종료 시간이 설정될 수 있습니다. 해당 시간 범위 안에서만 달란트를 받을 수 있습니다.

📍

위치 제한

위치 제한 QR은 브라우저 위치 권한을 허용해야 합니다. 지정 반경 밖이면 수령할 수 없으며, 위치 권한이 차단되어 있으면 알림창과 화면 메시지가 표시됩니다. 운영자는 100m부터 5km까지 반경을 선택할 수 있습니다.

🛒 학생 대신 상품 구매 신청하기

  1. 상품 구매 화면으로 이동합니다.
  2. 대리 구매 버튼을 누릅니다.
  3. 학생 이름이나 아이디로 대상을 검색하고 선택합니다.
  4. 선택한 학생이 구매할 수 있는 상품을 고릅니다.
  5. 확인 창에서 내용을 확인하고 구매 신청을 완료합니다.
대리 구매는 학생 대신 신청만 넣는 기능입니다. 실제 상품 준비, 구매 확정, 지급 처리는 구매 관리 권한이 있는 계정에서 진행합니다.
상품 카테고리, 대상(학생용/교사용), 구매 상태는 공통 코드 기준으로 표시됩니다. 대리 구매 대상자의 유형에 맞는 상품만 보이며, 신청 후 상태명은 내 구매 상품과 구매 관리에서 동일하게 표시됩니다.

⭐ 즐겨찾기 설정

메인 페이지의 바로가기 카드를 내가 원하는 메뉴로 설정할 수 있습니다.

1 메인 페이지에서 ⭐ 즐겨찾기 설정 버튼을 누릅니다.
2 내 권한에서 실제로 보이는 메뉴 중 1개 이상, 최대 10개까지 선택하고 저장을 누릅니다.
3 설정은 계정에 저장되어 다른 기기에서도 동일하게 표시됩니다.

🎨 테마, 상단 배지와 모바일 메뉴

테마 설정

상단 메뉴 오른쪽의 스위치로 일반/다크 모드를 전환합니다. 로그인 계정에 저장되며, 로그아웃하면 기본 일반 모드로 돌아갑니다.

처리 건수 배지

부서 담당 교사 이상은 관리/상품 메뉴에 처리 가능한 가입 신청, 부서 이동 요청, 구매 주문 건수가 표시됩니다.

모바일 네비게이션 (768px 이하)

화면 너비가 768px 이하인 모바일에서는 상단 바가 다음 순서로 정리됩니다.

  • 좌측: ⭐ 달란트 마을 (메인으로 이동)
  • 우측(왼쪽부터): 테마 스위치 → 햄버거 메뉴(☰) → 로그아웃 → (권한 이모지) 이름

햄버거 메뉴를 누르면 소개·달란트·상품·관리·운영 그룹이 아래로 펼쳐집니다. 모바일에서는 hover로 메뉴가 열리지 않으며, 그룹 이름을 탭해야 하위 메뉴를 볼 수 있습니다.

v3.51.0부터 소개 드롭다운의 여러 역할별 가이드 항목은 가이드 하나로 통합되며, 계정 권한에 맞는 가이드가 자동으로 열립니다. 관리(/admin/) 페이지에서도 동일한 모바일 상단 배치가 적용됩니다.

👥 부서 담당 교사 사용자 관리

부서 담당 교사(60+)는 관리 > 사용자 관리에서 담당 부서 소속 학생에 대해 다음 작업을 할 수 있습니다.

  • 반 변경: 관리 드롭다운 > 수정에서 반(class_number) 변경
  • 비밀번호 초기화: 관리 드롭다운 > 비밀번호 초기화 (초기값 1234)
사용자 삭제와 부서 이동은 부장 교사(80+) 이상만 가능합니다.
